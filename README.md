# Cloning the API
Make sure env file looks like this:
> DB_HOST=""<br>
> DB_USER=""<br>
> DB_NAME=""<br>
> DB_PASS=""<br>
> URL_LISTENING=""<br>
> RESTRICTED_IPS='["IP Here", "IP Here"]'
